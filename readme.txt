Questo plugin aiuta a personalizzare lo spazio hosting fornito da IWEBLAB.it .
Probabilmente il plugin non avrà effetto (e potrebbe creare malfunzionamenti) su altri hosting.


== Description ==
Con questo plugin puoi personalizzare alcune funzionalità offerte sugli hosting di IWEBLAB.it

== Installation ==
Per installare il plugin:

1. Decomprimi il file 'iweblab-hosting-manager.zip' e carica la cartella 'iweblab-hosting-manager' nella directory 'wp-content/plugins/' della tua installazione di WordPress.
2. Attiva il plugin dalla pagina "Plugins" del menù di WordPress.
3. Dal menù "Impostazioni" > "IWEBLAB Hosting Manager" puoi personalizzare le funzionalità previste. Appena attivato, il plugin mostra la configurazione di default del nostro hosting.
